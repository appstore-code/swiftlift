# SwiftLift Landing Page

Landing page for **SwiftLift** — the iPhone workout tracker with no account, no subscription, and no noise.

## Files

- `index.html` — Main landing page
- `style.css` — All styles
- `script.js` — Animations and interactions
- `support.html` — Support / FAQ page (use as your App Store Support URL)
- `privacy.html` — Privacy Policy (use as your App Store Privacy Policy URL)

## Setup on GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://appstore-code.github.io/swiftlift/`
