# SwiftLift Landing Page

Landing page for **SwiftLift** — the iPhone workout tracker with no account, no subscription, and no noise.

## Files

- `index.html` — Main landing page
- `style.css` — All styles
- `script.js` — Animations and interactions
- `support.html` — Support / FAQ page (use as your App Store Support URL)
- `privacy.html` — Privacy Policy (use as your App Store Privacy Policy URL)

## Setup on GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/swiftlift`

## App Store URLs to use

After publishing, use these URLs in App Store Connect:

- **Support URL:** `https://yourusername.github.io/swiftlift/support.html`
- **Privacy Policy URL:** `https://yourusername.github.io/swiftlift/privacy.html`
- **Marketing URL:** `https://yourusername.github.io/swiftlift`

## Customization

1. Replace `support@swiftlift.app` with your actual email address
2. Add your real App Store link to the download buttons (search for `href="#"` inside `<a class="btn-primary">` tags)
3. Replace the QR placeholder with a real QR code pointing to your App Store listing
4. Update copyright year if needed
